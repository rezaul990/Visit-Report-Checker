# RCM/RSM Visit Report (POS)

A modern, responsive web application for generating POS visit report links.

## Features

- 📅 Date range selection
- 👤 Employee ID input
- 🔗 Automatic link generation for each date
- 📱 Fully responsive design
- 🎨 Modern gradient UI

## Deployment

This app is ready to deploy to Netlify:

1. Push this repository to GitHub
2. Connect your GitHub repo to Netlify
3. Deploy (no build process needed - it's a static HTML app)

Or use Netlify CLI:
```bash
netlify deploy --prod
```
